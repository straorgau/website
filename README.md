# `stra/website`
This is the homepage for the [Student Transport Riders Association](https://stra.org.au), a student-run advocacy group in Melbourne, Australia.

Written with **Astro v5** & [TailwindCSS](https://tailwindcss.com).

## Instructions
**Prerequisites**: Install Git and [NodeJS](https://nodejs.org/en), preferably the latest LTS version.

```sh
$ corepack enable # Enable corepack, not required but strongly recommended for this repository
$ corepack install # Install the project's package manager (currently `pnpm`, see `package.json`)
$ pnpm install # Install project dependencies
$ pnpm dev # Run the development server, listens on http://localhost:4321 by default.
$ pnpm build # Build the production website
```

Automated builds on this branch are currently available at https://beta.stra.org.au.
